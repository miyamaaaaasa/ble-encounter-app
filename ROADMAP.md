# ROADMAP — はじめましてこんにちは

「Bluetoothですれ違った人との出会いを楽しむコミュニティゲーム」

最終更新: 2026-07-03 / 現行バージョン: **beta1.7.3+36**

---

## コンセプト（不変）

出会い / 偶然性 / ノスタルジー / コレクション / コミュニティ / ドット絵 / ワクワク感

### 絶対維持事項（破壊禁止）
- GPS不使用（BLEのみ）
- BLE中心設計
- 匿名性（本名・連絡先不要）
- 気配演出（すれ違い直後は相手を隠す）
- 開門システム（朝9/昼12/夜21の1日3回）
- トークン方式（回転する使い捨てトークン、永続IDを流さない）
- プライバシー保護（数値の抽象化・裏でのアドバタイズ停止）
